How Handles Work

In FHEVM, encrypted values are referenced by "handles" - unique identifiers pointing to encrypted data. Each operation creates a NEW handle. The original handles remain unchanged. Permissions must be set for each new handle using TFHE.allowThis() and TFHE.allow().
